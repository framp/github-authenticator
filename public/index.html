GitHub Authenticator

How to:

Create an application

Create or open your application settings on GitHub.

Set http:///callback/CLIENT_ID as an Authorization callback URL

Register your app 

  
  curl /apps -d"name=APPNAME&client_id=CLIENT_ID&client_secret=CLIENT_SECRET&callback=http://your-app.com/#{ID}"

Update your app 

  
  curl /apps -d"_method=PUT&name=APPNAME&client_secret=CLIENT_SECRET"

Delete your app 

  
  curl /apps -d"_method=DELETE&name=APPNAME&client_secret=CLIENT_SECRET"

Inspect your app 

  
  curl "/apps/CLIENT_ID?client_secret=CLIENT_SECRET"

Redirect your users

Send your user here: https://github.com/login/oauth/authorize?scope=user:email&client_id=CLIENT_ID .

The authenticator will redirect to the callback URL saved in the database with {ID} replaced with the access_token
